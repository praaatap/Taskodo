## Key Features

### Modern Timeline and Calendar
- **Intuitive Navigation**: Seamlessly swipe through months or jump back to Today with a single tap.
- **Visual Priority Indicators**: Days are marked with priority-colored dots for instant workload assessment.
- **Rich Agenda View**: A timeline-inspired daily list showing task progress, categories, and priority badges.

### Smart Task Organization
- **Category Grouping**: Instantly flip between a flat list and categorized headers with a single tap.
- **Subtask Mastery**: Break down complex objectives into manageable checklists.
- **Priority Driven**: Clear visual hierarchy to ensure critical tasks are addressed first.

### Focus and Habits
- **Daily Momentum**: Track streaks and celebrate wins with integrated habit tracking.
- **Focus Timer**: Dedicated deep-work sessions to maintain flow without distractions.
- **Mascot Companion**: A companion that celebrates successes and provides reminders for goals.

### Premium UI and UX
- **Glassmorphism**: Elegant backdrop blurs and frosted glass effects.
- **Fluid Animations**: Layout transitions that provide a professional feel.
- **Gesture Driven**: Intuitive swipe-to-complete and swipe-to-delete gestures.

## Tech Stack

- **Framework**: Expo (React Native)
- **Navigation**: Expo Router (File-based routing)
- **Animation**: React Native Reanimated and Gesture Handler
- **UI Components**: Custom Design System
- **Icons**: Feather Icons via @expo/vector-icons
- **Calendar**: React Native Calendars
- **Persistence**: AsyncStorage

## Getting Started

### Prerequisites
- Node.js (v18+)
- pnpm or npm
- Expo Go app on your mobile device (for testing)

## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/praaatap/Taskodo.git
   cd Taskodo
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   # or
   npm install
   ```

3. **Start the development server**
   ```bash
   npx expo start
   ```

4. **Run on your device**
   - Scan the QR code with your camera (iOS) or Expo Go app (Android).

## Design Philosophy

Taskodo follows a Clarity First philosophy. Using an Indigo and Violet color palette, the application prioritizes readability and visual excellence. Micro-animations provide meaningful feedback for user interactions.
